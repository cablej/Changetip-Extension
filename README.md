# Changetip-Extension

A Chrome extension that adds a Changetip "tip" button on websites such as Reddit, Soundcloud, and Twitter.

Currently supported:

* Twitter
* Reddit
* Soundcloud
* Slack

Support coming soon for:

* Youtube
* Stocktwits
